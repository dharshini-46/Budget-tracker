# Budget Tracker

A simple Budget Tracker web application to manage income and expenses.

## Features
- Add income and expense transactions
- Delete transactions
- Automatic calculation of income, expense, and balance
- Data persistence using localStorage
- Color-coded transactions
- Filter by All / Income / Expense
- Pie chart visualization of income vs expense

## Technologies Used
- HTML
- CSS
- JavaScript

## Setup
1. Clone the repository
2. Open index.html in a browser
